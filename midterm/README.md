# Identifying Wage Theft in New York
**This project was featured at Cooper Union's  *[Tracing the City](https://cooper.edu/events-and-exhibitions/exhibitions/tracing-city)* exhibit**

In close collaboration with Justicia Lab and Reclamo, this project presents an interactive exhibit that visualizes the scale of wage theft in New York. Visitors receive physical checks representing stolen wages and can scan them to access a webpage with details about the corresponding case and company.

Click here to view a sample [case](https://andrewtuna05.github.io/wage_theft.html?company=royal-care)!
